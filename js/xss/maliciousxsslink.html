This website could look entirely normal, but with a single line of JavaScript code, it can corrupt any webpage that linked to it.

window.opener.location.href does have a few uses, but generally it's only used for malicious purposes. However, not much can be done with it, as all it does is change the URL of the original page. However, bookmarklets are JavaScript codes linked to URLs and we can run JS urls to the original page.

It can also be used to run untrusted code on a trusted website. For example, this code javascript:(function()%7Bvar i %3D 2%3Bwhile (i <%3D 10) %7Bwindow.open("about:blank#gotcha"%2C""%2C"width%3D1200%2Cheight%3D600"%2C)%3Bi%20-%3D%202%3B%7D%7D)() spams pop-ups, but almost all pop-up blockers stop it from doing so. If you trusted a totally innocent site with popups, but that totally innocent site was hijacked by this one, then this code would pass through the pop-up blocker.
